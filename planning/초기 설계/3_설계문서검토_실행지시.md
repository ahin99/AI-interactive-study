`AI학습법공모전_설계문서2_ClaudeCode용.md`는 그대로 써도 된다. Claude Code에 넣을 첫 프롬프트, CLAUDE.md 초안, 타입, mock 데이터, 화면별 지시, Phase 체크리스트까지 이미 실행 가능한 수준으로 구체적이다. 다시 설계할 필요 없음.

## 오늘 할 일 순서

1. 이 프로젝트 폴더에서 Claude Code(`claude`)를 실행한다.
2. 위에서 만든 프롬프트를 첫 메시지로 붙여넣는다.
3. Claude Code가 docs 8개 + CLAUDE.md를 다 만들 때까지 기다리지 말고, 문서가 대략 끝나면 바로 Next.js 골격과 mock UI(Phase 1, 3, 4)로 넘어가라고 지시한다. 문서 8개를 완벽하게 다듬는 데 시간을 쓰지 마라.
4. 6단계 화면(홈→업로드→개념지도→백지복습→피드백→지식지도)이 mock 데이터로 끊김 없이 연결됐는지가 최우선 완료 기준이다. 나머지(디자인 디테일, 실제 API)는 그다음이다.
5. 앱이 돌아가면 `npm run dev`로 직접 브라우저에서 6단계를 처음부터 끝까지 클릭해서 확인한다. 특히:
   - 업로드 → 분석완료 → 개념지도 이동이 끊기지 않는지
   - 통정허위표시 백지복습 예시 답안 흐름이 시연 시나리오(문서 14번)와 맞는지
   - 피드백 화면에 정답 해설처럼 긴 설명이 나오지 않는지
6. 문제없으면 README 실행 방법을 확인하고, 배포 링크가 필요하면 Vercel 배포까지 진행한다.
7. 완성되는 대로 시연 영상 녹화하고, 저녁에 현이 보내주는 신청서 내용과 합쳐서 제출한다.
