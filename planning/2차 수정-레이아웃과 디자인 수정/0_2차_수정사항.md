# 요구사항

이 문서를 읽고 lexirecall에 있는 다음 사항 수정해줘. 내가 지시한 내용 외에는 최대한 현재 상황 덜 수정하는 방식으로.

1. 
- 지금은 첫 화면에 들어가서 예시 데이터가 '민법총칙' 만 있는데, 다음 추가해줘.
    - 헌법, 형법총론, 채권법각론, 물권법, 회사법
- 지금 있는 민법총칙이랑 같은 구조로 추가하고, 필요한 시연용 데이터는 임의로 생성해.

2. 첫 화면에 들어가서 최근 복습한 과목 나오는 부분 삭제해.

3. 첫 화면 구성 다음과 같이 바꿔.
- 상단에 Lexirecall에 대한 간략 설명 (화면의 상단 3분의 1 정도 크기로)
    - Lexirecall 로고
    - Lexirecall에 대한 간단한 개요 설명 (2줄 정도로.)
- 그 아래는 전체 과목을 띄워줘. 과목이 한 화면에 보이지 않으면 가로 스크롤로 볼 수 있게 해줘.
    - 숫자보다는 과목명을 더 강조해 가독성을 높여.
- 과목 추가 버튼 기능은 유지.

4. 과목에 들어갔을 때 화면 수정
- 과목명을 가장 강조해서 가독성을 높여.
- 개념 지도 디자인을 아래 형식으로 바꿔라. 지금은 너무 구려.
    - 주차별로 타일을 나누지 마라.
    - 1주차부터 16주차까지가 들어온다고 미리 상정해라. 그래서 주차별 칸 4*4가 고려된 정사각형 형태의 전체 개념 지도를 만들어라. 전체 지도는 정사각형이고, 각 주차도 정사각형이다. 모든 칸은 정사각형이다.
        - 개념 지도는 이미지 참고 폴더 내의 이미지를 참고해 비슷한 느낌으로 만들어.
        - 개념 지도 참고 1 의 이미지를 참고해라. 지금처럼 주차 간 타일을 화면에 구분해 표시하지 마라. 희미한 회색 선으로만 구분하고, 클릭하면 그 주차로 들어가는 식으로 구성해라.
        - 개념 지도 참고 2, 개념 지도 참고 3의 이미지처럼 최종적으로 이런 미감으로 나왔으면 좋겠어. 전체를 16주차로 가정하고 자료가 입력되지 않은 곳은 하얀색으로 비워놔.


        - 클릭하면 지금처럼 기존 페이지 밑에 해당 주차를 띄우는 게 아니라, 새로운 페이지를 띄운다. 
            - 왼쪽은 정사각형으로 된 확대된 상태의 해당 주차 개념 지도
            - 개념 지도 바로 아래에 '백지복습 하기' 버튼을 가독성 좋게 검은색으로 만들어.
            - 오른쪽 화면은 이전 백지복습 기록을 볼 수 있게 띄워준다. 
                - 백지복습을 여러 번 해서 스크롤이 필요한 경우는 세로 스크롤.

- 가장 눈에 들어오는게 개념 지도여야 해. 
    - 가운데 화면에 개념 지도를 크게 띄워라. 화면 가운데를 전체 차지하게.
    - 지식 지도/메타인지 지도 전환 버튼, 그리고 스크롤은 화면 맨 하단으로 내려.
    - 지도 완성률, 메타인지 오차율은 과목명 바로 아래 상단 바 형식으로 작게 붙여라. 타일 형식 말고.


5. 백지복습 하기 버튼을 누르고 나서
    - 상단 바에 진행 과정을 띄운다.
        - 1. 개념 체크 -> 2. 백지 복습 -> 3. 피드백
    
    - 개념 체크 단계에서는 알고 있다고 생각하는 개념 체크를 한다.
        - 지금처럼 단순 체크박스 형태가 아니라, 노션 토글처럼 개념을 하위 개념으로 분류해서 가독성 높여.
    
    - 즉 개념 체크와 백지 복습이 지금은 한 페이지에 있는데 그걸 두 단계로 분리하라는 말이다.

6. 피드백 부분
- 지금 내 답안에서 내가 입력한 줄의 바로 밑줄에 피드백을 작성하라고 했어. 즉 한줄한줄 쓰라는 말이야. 근데 지금 내 답안 타일에 피드백 문장이 아무것도 없고 형광펜은 있는데 내가 쓴 문장만 있지? 이러면 안돼.
- 내 답안, 누락된 내용 단서, 복습 제안 한줄은 왼쪽 화면에 배치.
    - 최종 복습 제안 한줄에는 '복습 제안' 이라고 이름 붙여. 지금 이거 한줄 나온 말이 무슨 말인지 의도를 모르겠어.
- 오른쪽 화면에는 소크라테스식 이해 검증 질문과 결과 저장 버튼을 두기. 다시 복습 버튼은 없애.

7. 전체적으로, 너무 간단한 디자인을 유지하려고 해서 미감이 구려짐.
